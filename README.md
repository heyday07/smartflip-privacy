# SmartFlip Privacy Policy

This repository hosts the privacy policy for **SmartFlip** (Smart Trivia / 매일매일 상식퀴즈 / デイリー雑学クイズ), an Android trivia learning app by RedMoon Studio.

## Live Page

🌐 **https://heyday07.github.io/smartflip-privacy/**

The page auto-detects browser language and supports manual switching between:
- 🇰🇷 한국어 (Korean)
- 🇺🇸 English
- 🇯🇵 日本語 (Japanese)

You can also force a specific language via URL query: `?lang=ko`, `?lang=en`, `?lang=ja`
