개인정보처리방침

SmartFlip (매일매일 상식퀴즈) | 시행일: 2026년 5월 6일 | 버전 1.0

RedMoon Studio(이하 "회사")는 「개인정보 보호법」 제30조에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리하기 위하여 다음과 같이 개인정보처리방침을 수립·공개합니다.

한 줄 요약: SmartFlip은 회원가입이 없으며, 어떤 개인정보도 외부로 전송하지 않습니다. 모든 학습 데이터는 사용자의 기기에만 저장됩니다.

제1조 (수집하는 개인정보 항목)

1. 수집하지 않는 정보

SmartFlip은 다음 정보를 일체 수집하지 않습니다.

  • 이름, 이메일, 전화번호, 생년월일 등 개인식별정보
  • 위치 정보(GPS)
  • 연락처, 통화 기록
  • 광고 식별자(Android Advertising ID)
  • 앱 사용 통계, 기기 정보, 네트워크 정보
  • 충돌(Crash) 데이터, 분석 데이터

2. 기기 내부에만 저장되는 데이터

다음 데이터는 사용자의 기기 저장소에만 보관되며, 외부 서버로 전송되지 않습니다.

항목 용도 저장 위치
학습 진도(SRS 데이터) 간격 반복 학습 알고리즘 로컬 SQLite
사용자가 만든 카드 학습 콘텐츠 로컬 SQLite
사용자가 첨부한 카드 이미지 플래시카드 시각 자료 앱 전용 저장소
퀴즈 기록, 통계, 게이미피케이션 데이터 학습 동기 부여 로컬 SQLite
설정 정보(언어, 테마, 알고리즘 등) 앱 환경 설정 SharedPreferences

제2조 (앱 권한 사용 목적)

SmartFlip은 다음 권한을 다음 목적으로만 사용합니다.

권한 사용 목적 필수 여부
카메라 (Camera) 카드 만들기 시 이미지 촬영 (선택 기능) 선택
사진 / 미디어 (READ_MEDIA_IMAGES) 갤러리에서 카드 이미지 선택 (선택 기능) 선택
인터넷 (INTERNET) v1.0에서는 사용하지 않음 (Android 표준 요구사항) 선언만 존재

카메라 또는 사진 권한을 거부하셔도 앱의 모든 학습 기능은 정상적으로 사용 가능합니다(이미지 첨부만 비활성).

제3조 (개인정보의 처리 목적)

회사는 외부로 어떠한 개인정보도 전송하지 않으며, 사용자의 데이터를 다음 목적으로만 기기 내에서 처리합니다.

  1. 학습 기능 제공: 플래시카드 학습, 퀴즈, 간격 반복 학습 알고리즘
  2. 사용자 콘텐츠 관리: 사용자가 만든 카드의 저장, 검수, 학습
  3. 학습 통계 표시: 사용자 본인의 학습 진도와 성과 시각화
  4. 게이미피케이션: EXP, 레벨, 데일리 미션, 스트릭 뱃지

제4조 (개인정보의 제3자 제공)

회사는 사용자의 개인정보를 제3자에게 제공하지 않습니다. SmartFlip v1.0은 외부 서버와 일체의 통신을 하지 않습니다.

제3자 서비스(SDK)

SmartFlip v1.0은 어떤 제3자 분석/광고 SDK도 사용하지 않습니다.

향후 버전(v1.1 이상)에서 광고(Google AdMob), 분석(Firebase Analytics), 충돌 보고(Firebase Crashlytics) 등이 추가될 경우, 본 개인정보처리방침을 갱신하고 앱 업데이트를 통해 사전 공지합니다.

제5조 (사용자가 선택적으로 공유하는 데이터)

다음은 사용자의 명시적 액션이 있을 때만 처리됩니다.

  • .smartflip 파일 공유: 사용자가 만든 카드 세트를 다른 사람과 공유할 때, 사용자가 직접 공유 메뉴(카카오톡, 이메일 등)를 통해 파일을 전송합니다. 이 파일에는 카드 내용만 포함되며, 학습 진도 등 개인 데이터는 포함되지 않습니다.
  • .sfbackup 파일 백업/복원: 사용자가 직접 백업 파일을 만들어 외부 저장소에 보관/복원할 수 있습니다. 회사는 이 과정에 관여하지 않습니다.
  • 피드백 이메일: 설정 > 문의/피드백을 통해 사용자가 직접 이메일을 작성해 보내는 경우, 사용자가 입력한 내용만 회사에 전달됩니다.
주의: .smartflip 또는 .sfbackup 파일을 공유하기 전, 카드에 개인정보(이름, 전화번호 등)가 포함되어 있지 않은지 확인해주세요. 파일이 외부로 나간 이후의 관리는 사용자의 책임입니다.

제6조 (개인정보의 보유 및 파기)

회사는 사용자의 데이터를 직접 저장하지 않습니다. 모든 데이터는 사용자의 기기에 저장되며, 다음 방법으로 삭제할 수 있습니다.

  • 앱 삭제: 모든 학습 데이터, 사용자 카드, 설정이 즉시 영구 삭제됩니다.
  • 앱 내 데이터 초기화: 설정 메뉴에서 카드, 학습 진도를 개별 삭제 가능합니다.
  • 휴지통 자동 정리: 삭제된 카드는 30일 후 자동 영구 삭제됩니다.

제7조 (이용자의 권리)

이용자는 다음과 같은 권리를 행사할 수 있습니다.

  1. 데이터 열람: 앱 내 통계 화면에서 본인의 모든 학습 데이터 확인 가능
  2. 데이터 수정/삭제: 앱 내 카드 관리 화면에서 직접 수정/삭제 가능
  3. 데이터 백업: .sfbackup 파일로 사용자 데이터 export 가능
  4. 전체 데이터 삭제: 앱 삭제로 모든 데이터를 즉시 삭제

제8조 (아동의 개인정보 보호)

SmartFlip은 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 본 앱은 외부로 어떤 정보도 전송하지 않으므로 아동의 개인정보가 외부로 유출될 가능성이 없습니다.

제9조 (개인정보의 안전성 확보 조치)

  • 모든 사용자 데이터는 기기 내부에만 저장 (외부 서버 없음)
  • 이미지 파일은 앱 전용 저장소에 보관 (다른 앱이 접근 불가)
  • 카드 입력 시 자동 sanitize (HTML/스크립트 제거, XSS 방지)
  • 민감정보 패턴 자동 감지 및 경고 (주민번호, 카드번호, SSN, 마이넘버 등)
  • 파일 공유 전 민감정보 경고 다이얼로그

제10조 (개인정보처리방침의 변경)

본 개인정보처리방침이 변경되는 경우, 변경 사항은 앱 업데이트 또는 본 페이지를 통해 공지합니다. 중대한 변경(예: 외부 서비스 SDK 추가)이 있을 경우 변경 사항이 효력을 발생하기 7일 전 사전 공지합니다.

Privacy Policy

SmartFlip (Smart Trivia) | Effective Date: May 6, 2026 | Version 1.0

RedMoon Studio ("Company", "we", "us") operates the SmartFlip mobile application. This Privacy Policy explains how we handle information when you use our app.

One-line Summary: SmartFlip does not require account registration and does not transmit any personal information externally. All learning data is stored only on your device.

What We Do NOT Collect

SmartFlip does NOT collect any of the following:

  • Personal identifiers such as name, email, phone number, date of birth
  • Location data (GPS)
  • Contacts or call history
  • Advertising identifiers (Android Advertising ID)
  • App usage statistics, device information, network information
  • Crash data or analytics data

Data Stored Only On Your Device

The following data is stored exclusively on your device's storage and is never transmitted to any external server:

Data Type Purpose Storage Location
Learning progress (SRS data) Spaced repetition algorithm Local SQLite
User-created cards Learning content Local SQLite
User-attached card images Flashcard visual aids App-private storage
Quiz history, statistics, gamification data Learning motivation Local SQLite
Settings (language, theme, algorithm, etc.) App configuration SharedPreferences

2. App Permissions

SmartFlip requests the following permissions only for the stated purposes:

Permission Purpose Required
Camera Take photos when adding card images (optional feature) Optional
Photos / Media (READ_MEDIA_IMAGES) Select gallery images for cards (optional feature) Optional
Internet Not used in v1.0 (Android standard requirement) Declared only
Notifications (POST_NOTIFICATIONS) Disabled in v1.0 (study reminders coming in v1.1) Optional

Even if you deny camera or photo permissions, all learning features will work normally (only image attachment is disabled).

3. How We Use Information

We do not transmit any personal information externally. Your data is processed only on your device for the following purposes:

  1. Learning Features: Flashcard study, quizzes, spaced repetition algorithm
  2. User Content Management: Saving, reviewing, and studying user-created cards
  3. Statistics Display: Visualizing your own learning progress and performance
  4. Gamification: EXP, levels, daily missions, streak badges

4. Third-Party Sharing

We do not share any personal information with third parties. SmartFlip v1.0 does not communicate with any external server.

Third-Party Services (SDKs)

SmartFlip v1.0 does NOT use any third-party analytics or advertising SDKs.

If future versions (v1.1 or later) add advertising (Google AdMob), analytics (Firebase Analytics), or crash reporting (Firebase Crashlytics), this Privacy Policy will be updated and notified through an app update in advance.

5. Data You Voluntarily Share

The following data is processed only when you take explicit action:

  • .smartflip file sharing: When you share a card set with others, you choose the share target (KakaoTalk, email, etc.) yourself. The file contains only card content; no personal data such as learning progress is included.
  • .sfbackup file backup/restore: You can create backup files yourself and store/restore them from external storage. We are not involved in this process.
  • Feedback email: When you write an email through Settings > Feedback, only the content you type is sent to us.
Caution: Before sharing .smartflip or .sfbackup files, please verify that no personal information (name, phone number, etc.) is included in the cards. You are responsible for managing the file once it leaves your device.

6. Data Retention and Deletion

We do not store any user data ourselves. All data is stored on your device and can be deleted as follows:

  • App uninstallation: All learning data, user cards, and settings are immediately and permanently deleted.
  • In-app data clearing: Cards and learning progress can be deleted individually from the settings menu.
  • Trash auto-cleanup: Deleted cards are permanently removed after 30 days.

7. Your Rights

  1. Data access: View all your learning data in the in-app statistics screen
  2. Data modification/deletion: Edit/delete cards directly in the card management screen
  3. Data backup: Export your data as .sfbackup file
  4. Complete data deletion: Uninstall the app to immediately delete all data

8. Children's Privacy

SmartFlip does not knowingly collect personal information from children under the age of 13. Since this app does not transmit any information externally, there is no possibility of children's personal information being leaked.

9. Security

  • All user data is stored only on the device (no external server)
  • Image files are kept in app-private storage (inaccessible to other apps)
  • Automatic sanitization of card input (HTML/script removal, XSS prevention)
  • Automatic detection and warning of sensitive information patterns (Resident ID, card number, SSN, My Number, etc.)
  • Sensitive information warning dialog before file sharing

  3. データバックアップ: .sfbackupファイルでユーザーデータをエクスポート可能
